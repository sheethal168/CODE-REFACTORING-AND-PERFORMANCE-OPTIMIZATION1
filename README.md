# Expense Tracker (Refactored & Optimized)

## 📌 Project Overview
This project is a simple Expense Tracker web application that allows users to add expenses, view them in a list, and calculate the total amount spent. The project was originally taken from an open-source GitHub repository and improved through code refactoring and performance optimization.

---

## 🚀 Features
- Add expenses with name and amount
- Display expense list dynamically
- Calculate and update total amount
- Delete expenses from the list
- Clean and responsive user interface

---

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

---

## 🔧 Improvements Made

### 🔹 Code Refactoring
- Renamed variables with meaningful names
- Split large logic into smaller functions:
  - `addExpense()`
  - `renderExpenses()`
  - `deleteExpense()`
- Improved code structure and readability
- Removed redundant code

### 🔹 Performance Optimization
- Cached DOM elements to avoid repeated access
- Reduced unnecessary operations
- Optimized rendering logic
- Added input validation

### 🔹 UI Improvements
- Fixed alignment issues
- Improved layout using Flexbox
- Enhanced overall user interface
